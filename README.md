# StoreOL
基于SSH框架开发的一款在线购物网站


项目简介：
前端框架使用的主要是easyui和jquery

后端框架使用的主要是Spring4、Struts2、Hibernate5

数据库使用的mysql

服务器用的是tomcat

开发工具用的是eclipse

主要实现的功能是顾客的注册、登录和修改个人信息；实现了商品的分类、购买，以及购物车和订单详情等功能；
在未登录时实现权限管理，使其不能点击购物、购物车、订单详情等与顾客相关的功能；
